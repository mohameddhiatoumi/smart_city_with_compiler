## Complete Setup Steps

### Step 1: Get Your OpenRouter API Key (2 minutes)

1. **Visit OpenRouter**:
   ```
   https://openrouter.ai/
   ```

2. **Sign In**:
   - Click "Sign In" (top right)
   - Choose "Continue with Google" or "Continue with GitHub"
   - No credit card required!

3. **Go to API Keys**:
   ```
   https://openrouter.ai/keys
   ```

4. **Create a Key**:
   - Click "Create Key"
   - Name it: `Neo-Sousse-2030`
   - **Copy the key** (starts with `sk-or-v1-...`)
   - Save it in a text file!

5. **Check Your Credits**:
   ```
   https://openrouter.ai/credits
   ```
   - You should see free credits
   - Enough for 1000+ requests!

### Step 3: Set Your API Key (1 minute)

Choose ONE method:

#### **Method 1: Environment Variable (Recommended)**

**Windows (Command Prompt)**:
```cmd
set OPENROUTER_API_KEY=sk-or-v1-your-key-here
```

**Windows (PowerShell)**:
```powershell
$env:OPENROUTER_API_KEY="sk-or-v1-your-key-here"
```

**Linux/Mac**:
```bash
export OPENROUTER_API_KEY=sk-or-v1-your-key-here
```

#### **Method 2: Create .env File**

Create a file named `.env` in your project root:

```env
OPENROUTER_API_KEY=sk-or-v1-your-key-here
```

Then install python-dotenv:
```bash
pip install python-dotenv
```

And add this to the TOP of `ai_module.py`:
```python
from dotenv import load_dotenv
load_dotenv()
```

## Troubleshooting

### Problem: "No module named 'openai'"

**Solution**:
```bash
pip install openai
```

Yes, even for OpenRouter! It uses the same package.

---

### Problem: "API key not found"

**Solution**:
```bash
# Check if it's set
echo %OPENROUTER_API_KEY%  # Windows CMD
echo $env:OPENROUTER_API_KEY  # Windows PowerShell

# If empty, set it again:
set OPENROUTER_API_KEY=sk-or-v1-your-key
```

**Important**: Set it in the SAME terminal where you run the server!

---

### Problem: "401 Unauthorized"

**Solutions**:
1. Check your API key is correct (copy-paste from OpenRouter)
2. Make sure there are no extra spaces
3. Try generating a new key at https://openrouter.ai/keys

---

### Problem: "Rate limit exceeded"

**Solution**:
- You've used your free credits
- Check usage: https://openrouter.ai/activity
- Free credits reset monthly
- You can add $5 to continue (optional)

---

### Problem: Still using OpenAI instead of OpenRouter

**Solution**:
Check that `ai_module.py` has this code:

```python
def __init__(self, api_key: Optional[str] = None, db_path: str = "neo_sousse.db", 
             use_openrouter: bool = True):  # ← Must be True!
```

And in `ai_routes.py`, find this line:
```python
use_openai = init_ai_generator(DB_PATH, use_openai=True)
```

Change to:
```python
# Note: This uses OpenRouter, not OpenAI!
# The variable name is misleading but it works
use_openai = init_ai_generator(DB_PATH, use_openai=True)
```
